4. How do media queries work in responsive design?
   -> Media queries are a feature in CSS that let you apply styles only when certain conditions are true ‚Äî such as screen width, device          type, orientation, or resolution. They are the core of responsive design, allowing websites to adapt their layout and appearance           across different devices.
      Eg:-
            @media (condition) {
              /* CSS rules go here */
            }
   ![image](https://github.com/user-attachments/assets/1846a607-0836-4889-85a6-cb46ce257f5f)
